## ⚡ START HERE - 3 SIMPLE STEPS

### 1️⃣ Setup Midtrans (5 min)
```
✓ Go to https://midtrans.com
✓ Sign up → Business
✓ Copy Server Key & Client Key dari Settings → API Keys
✓ Save to temporary text file
```

### 2️⃣ Deploy Backend to Vercel (5 min)
```bash
npm install -g vercel
cd backend
vercel --prod --name disba-backend
# When asked, input your Midtrans keys and Supabase credentials
# SAVE the backend URL shown (e.g., https://disba-backend.vercel.app)
```

### 3️⃣ Deploy Frontend to Vercel (5 min)
```bash
cd frontend
vercel --prod --name disba-music
# Set VITE_API_URL = your backend URL from step 2
# SAVE the frontend URL shown
```

## 🛠️ TROUBLESHOOTING

### Payment button doesn't appear?
→ Check VITE_API_URL in frontend environment  
→ Verify backend URL is correct

### Midtrans modal doesn't show?
→ Open browser console (F12)  
→ Check for errors  
→ Verify window.snap exists

### Database connection error?
→ Check SUPABASE_SERVICE_ROLE_KEY  
→ Verify Supabase project is active

### Webhook not triggering?
→ Setup webhook in Midtrans Dashboard  
→ URL: https://your-api.com/api/payments/notification  
→ Check backend logs
